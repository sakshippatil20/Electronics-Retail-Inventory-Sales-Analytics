# 📊 Electronics Retail Inventory & Sales Analytics (Power BI)

## 🔎 Project Overview
This Power BI project analyzes sales performance and inventory management for an electronics retail business.  
The dashboard provides interactive insights into revenue trends, stock levels, reorder requirements, and product performance.

The goal is to support data-driven decision-making using KPIs and DAX-based calculations.

---

## 🎯 Business Objectives
- Monitor total revenue and order performance
- Track stock availability and low-stock alerts
- Identify products requiring reorder
- Analyze category and brand performance
- Improve inventory turnover efficiency

---

## 🛠 Tools & Technologies Used
- Power BI Desktop
- Data Modeling
- DAX (Data Analysis Expressions)
- KPI Cards
- Slicers & Filters
- Interactive Visualizations

---

## 📊 Key KPIs
- Total Revenue
- Total Orders
- Inventory Turnover
- Low Stock Products
- Reorder Priority Items
- Category-wise Sales Performance
